# Code of Conduct

## Our Pledge

We are committed to making participation in this project a respectful, inclusive, and harassment-free experience for everyone.

This project follows a simple rule: **be respectful and constructive**.

---

## Expected Behavior

Contributors are expected to:

- Be respectful and considerate in discussions
- Give and accept constructive feedback
- Focus on what is best for the community and the project
- Use welcoming and inclusive language
- Assume good intentions

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or hate speech
- Personal attacks, insults, or trolling
- Intimidation or threats
- Publishing private information without consent
- Any behavior that would reasonably make others uncomfortable

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub repositories (issues, pull requests, discussions)
- Project documentation
- Any official communication channels related to the project

---

## Enforcement

Maintainers are responsible for clarifying standards of acceptable behavior and taking appropriate action in response to any instances of unacceptable behavior.

Actions may include:

- Warning the offender
- Temporary or permanent ban from the project
- Rejection of contributions that violate this Code of Conduct

---

## Reporting

If you experience or witness unacceptable behavior, please report it to the project maintainers.

Reports will be reviewed and handled discreetly and respectfully.

---
